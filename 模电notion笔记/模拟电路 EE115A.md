## 🧠 总体总结概括

在进入器件与电路之前，课程先用 **基础篇（L1–L6）** 打地基：L1 讲模拟电子学的意义与 CMOS 工艺；L2 把放大器抽象为增益 / 带宽 / 二端口模型 / 频率响应；L3–L4 用「虚短 + 虚断」建立理想运放，再补上失调电压、有限增益带宽、压摆率、CMRR 等非理想；L5–L6 转入半导体物理（载流子、PN 结 $I=I_S(e^{V/V_T}-1)$），为 L7 起的 MOSFET 器件做铺垫。在此基础上：

模拟电路的主线是 **从单管到系统** 的逐步搭建，期末复习课（[EE115 Lecture17 — 期末复习：MOSFET → 单级 → 差分 → 运放 → 反馈](%E6%A8%A1%E6%8B%9F%E7%94%B5%E8%B7%AF%20EE115A/EE115%20Lecture17%20%E2%80%94%20%E6%9C%9F%E6%9C%AB%E5%A4%8D%E4%B9%A0%EF%BC%9AMOSFET%20%E2%86%92%20%E5%8D%95%E7%BA%A7%20%E2%86%92%20%E5%B7%AE%E5%88%86%20%E2%86%92%20%E8%BF%90%E6%94%BE%20%E2%86%92%20%E5%8F%8D%E9%A6%88.md)）把它收成 5 个支柱：

1. **器件层**：MOSFET 三个工作区 + 饱和区小信号模型（$g_m$、$r_o$、本征增益 $A_0=g_m r_o$）是一切分析的原子。
2. **单级放大器**：共源 / 共栅 / 共漏 三种拓扑，及其阻抗变换规律（源端阻抗 $\times A_0$、漏端负载 $\div A_0$）。
3. **偏置 & 电流镜**：电流镜按 $W/L$ 比提供匹配偏置电流；理想电流源负载把单管增益推到本征增益 $A_0$。
4. **复合结构**：cascode（上下都堆 → $A_0^2$）与差分对（虚地 + 半电路 + 电流镜负载双端转单端），在高输出阻抗、共模抑制两个维度上组合出高性能电路。
5. **系统层面**：两级 CMOS 运放（差分增益级 + 共源增益级 + Miller 补偿，$A_o=A_1A_2$）是模拟信号链「原型」；外面再套 **负反馈** $A_f=\dfrac{A}{1+A\beta}$，用去敏因子 $(1+A\beta)$ 同时换来稳增益 / 展带宽 / 降失真 / 抑干扰。

<aside>
🎯

**四个贯穿本门课的「口诀」：**

1. 小信号分析 = 「确认饱和区 + 画出 $g_m, r_o$ + KCL/KVL」。
2. 高增益 ⇔ 高输出阻抗（$A_v = g_m R_{out}$），cascode 是提高 $R_{out}$ 的万能手段（单端可达 $A_0^2$）。
3. 差分结构同时解决 **偏置漂移**、**共模干扰**、**信号失调** 三大实际问题。
4. 负反馈 = 用增益换性能：$A_f=\dfrac{A}{1+A\beta}$，$(1+A\beta)$ 倍地稳增益 / 展带宽 / 降失真 / 抑干扰。
</aside>

## 📖 专业词速查（Glossary）

> 课程里遇到的专业英文 / 术语随手记在这里，方便日后查阅。
> 

| 术语 | 含义 | 公式 / 备注 |
| --- | --- | --- |
| **Short-circuit transconductance** $G_m$（短路跨导） | 把放大器输出端「短路」（$v_O=0$，即 $R_L=0$）时，输出短路电流与输入电压之比，衡量该级「产生信号电流」的能力，与输出电阻 $R_o$ 解耦。 | $G_m=\dfrac{i_{o,\text{sc}}}{v_{in}}$；整级 $A_d=G_m R_o$（接负载 $A_d=G_m(R_o\|R_L)$）；差分对+电流镜负载时 $G_m=g_{m1,2}$（满 $g_m$）。 |
| **Desensitivity factor** $(1+A\beta)$（去敏因子 / 反馈量，amount of feedback） | 负反馈把「闭环增益对开环增益变化的灵敏度」除掉的那个因子，也等于开环增益被反馈压低的倍数；值越大，闭环增益越稳定、越不受 $A$ 漂移影响。 | $A_f=\dfrac{A}{1+A\beta}$；灵敏度 $\dfrac{dA_f/A_f}{dA/A}=\dfrac{1}{1+A\beta}$；数值上 $1+A\beta=A/A_f$。 |
| **Intrinsic gain** $A_0$（本征增益） | 单管在理想电流源负载下能拿到的最大电压增益，是衡量工艺「增益能力」的基准。 | $A_0=g_mr_o=\dfrac{2|V_A|}{V_{OV}}=\dfrac{2V_A'L}{V_{OV}}$；$\propto L$、反比 $V_{OV}$。 |
| **Cascode**（共源-共栅堆叠） | 用共栅管把输出阻抗抬高 $A_0$ 倍，从而把增益从 $A_0$ 推到 $A_0^2$；要拿全增益必须上下负载都 cascode。 | $R_o\simeq A_0 r_o$；单端 $A_{vo}=-A_0^2$，对称 cascode 负载 $-\tfrac12 A_0^2$。 |
| **Virtual ground**（虚地） | 差分对在差分激励下反对称，公共源点电位不动、等效交流接地 —— 无需大 bypass 电容即可做半电路分析。 | 半电路 $A_d=g_m(R_D\|r_o)$；省电容 → 面积小、频响好。 |
| **ICMR / CMRR**（输入共模范围 / 共模抑制比） | ICMR 是保证尾电流源与输入管都在饱和区的共模输入区间；CMRR 衡量差模相对共模增益的优势。 | $V_{CM,\max}=V_t+V_{DD}-\tfrac{I}{2}R_D$，$V_{CM,\min}=-V_{SS}+V_{CS}+V_t+V_{OV}$。 |
