## 🧩 详细防错卡（每题 2–3 行）

[把总尾电流误当成单管电流](EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/%E6%8A%8A%E6%80%BB%E5%B0%BE%E7%94%B5%E6%B5%81%E8%AF%AF%E5%BD%93%E6%88%90%E5%8D%95%E7%AE%A1%E7%94%B5%E6%B5%81.md)

- **错在哪**：把 tail current $I$ 当成单只 MOS 的 $I_D$，忘记平衡时每边只有 $I/2$。
- **防错重点**：differential pair 题先标清 total tail current vs single-branch current；算 $g_m$ / $W/L$ 用单管 $I_D$。

[把直流偏置电流和小信号输出电阻混在一起](EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/%E6%8A%8A%E7%9B%B4%E6%B5%81%E5%81%8F%E7%BD%AE%E7%94%B5%E6%B5%81%E5%92%8C%E5%B0%8F%E4%BF%A1%E5%8F%B7%E8%BE%93%E5%87%BA%E7%94%B5%E9%98%BB%E6%B7%B7%E5%9C%A8%E4%B8%80%E8%B5%B7.md)

- **错在哪**：把题目给定的 DC current 误解成还会被有限 $r_o$ 分掉一部分。
- **防错重点**：DC bias 直接用题目给的电流；$r_o$ 只描述小信号增量电流随端口电压变化。

[把局部输入阻抗和输出负载重复计入](EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/%E6%8A%8A%E5%B1%80%E9%83%A8%E8%BE%93%E5%85%A5%E9%98%BB%E6%8A%97%E5%92%8C%E8%BE%93%E5%87%BA%E8%B4%9F%E8%BD%BD%E9%87%8D%E5%A4%8D%E8%AE%A1%E5%85%A5.md)

- **错在哪**：把局部 $R_{in}$ 当成严格 loaded input resistance，硬把输出负载混入，破坏分块估算。
- **防错重点**：先判断题目问的是局部量还是整体量；输出负载通常留到 $R_o$ 里统一处理。

[把电源轨误当成输入电压](EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/%E6%8A%8A%E7%94%B5%E6%BA%90%E8%BD%A8%E8%AF%AF%E5%BD%93%E6%88%90%E8%BE%93%E5%85%A5%E7%94%B5%E5%8E%8B.md)

- **错在哪**：把 power supplies / rails 误读成 gate 或 input voltage，导致不会列电源和功耗方程。
- **防错重点**：supplies / rails 是电源轨；DC 保留，小信号接 AC ground，功耗看 $(V_{DD}-V_{SS})I$。

[漏把功耗限制转成偏置电流方程](EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/%E6%BC%8F%E6%8A%8A%E5%8A%9F%E8%80%97%E9%99%90%E5%88%B6%E8%BD%AC%E6%88%90%E5%81%8F%E7%BD%AE%E7%94%B5%E6%B5%81%E6%96%B9%E7%A8%8B.md)

- **错在哪**：觉得题目少方程，没把 power dissipation 转成 tail current 约束。
- **防错重点**：设计题缺方程时扫 spec；功耗、增益、DC 输出点、saturation/headroom 都可能给方程。

[无依据引入尾电阻导致模型跑偏](EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/%E6%97%A0%E4%BE%9D%E6%8D%AE%E5%BC%95%E5%85%A5%E5%B0%BE%E7%94%B5%E9%98%BB%E5%AF%BC%E8%87%B4%E6%A8%A1%E5%9E%8B%E8%B7%91%E5%81%8F.md)

- **错在哪**：没有图时主动假设存在要设计的 $R_{SS}$，把 tail current source 和 tail resistor 混在一起。
- **防错重点**：先看题目 specify 什么；没写 tail resistance / source resistance，就不要自己加 $R_{SS}$。

[英文术语没有先翻成电路动作](EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/%E8%8B%B1%E6%96%87%E6%9C%AF%E8%AF%AD%E6%B2%A1%E6%9C%89%E5%85%88%E7%BF%BB%E6%88%90%E7%94%B5%E8%B7%AF%E5%8A%A8%E4%BD%9C.md)

- **错在哪**：dissipate / equilibrium 读题不稳，容易把功耗条件和平衡工作点割裂。
- **防错重点**：dissipate 看电路从电源取能并耗散；equilibrium 在差分对里就是两边对称、各分 $I/2$。

[混淆 PMOS/NMOS 与 S/G/D 端子](EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/EE115A%20%E9%94%99%E9%A2%98%E6%9C%AC/%E6%B7%B7%E6%B7%86%20PMOS%20NMOS%20%E4%B8%8E%20S%20G%20D%20%E7%AB%AF%E5%AD%90.md)

- **错在哪**：分析 MOS 电路时没先确认 PMOS / NMOS 类型与 source / gate / drain 位置，导致 $V_{GS}$ / $V_{SG}$、电流方向或饱和条件写错。
- **防错重点**：下笔前固定三步：①判 P/N MOS；②在图上标 S/G/D；③再写 $V_{OV}$、饱和条件和小信号模型。
