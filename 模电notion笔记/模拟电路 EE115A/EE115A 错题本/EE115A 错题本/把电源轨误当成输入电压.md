# 把电源轨误当成输入电压

错误总结: supplies / rails 是上下电源边界；DC 分析保留，小信号分析接 AC ground，功耗用电源跨度乘静态电流。
错误类型: 📋 审题漏条件, 🔀 概念混淆

**❌ 错在哪**：把 “±1-V power supplies” 误读成 gate/input voltage，导致不知道如何列 $V_{DD}$、$V_{SS}$ 和功耗方程。

**✅ 正确做法**：±1-V supplies 是电源轨：$V_{DD}=+1\,\text{V}$，$V_{SS}=-1\,\text{V}$；DC 分析保留，小信号分析视为 AC ground，功耗用 $(V_{DD}-V_{SS})I$。

**🐈 防错提醒**：看到 supplies / rails / $V_{DD}$ / $V_{SS}$，先画成电路上下边界，不要当输入信号。
