# 无依据引入尾电阻导致模型跑偏

错误总结: 没有写 tail resistance / source resistance 时，不要自己加 RSS；按题目给的 tail current source 处理。
错误类型: 🔀 概念混淆

**❌ 错在哪**：没有图时主动假设存在需要设计的 $R_{SS}$，把 ideal tail current source 和 tail resistor 混在一起。

**✅ 正确做法**：题目要求 specify $I, R_D, W/L$，说明 tail 支路按提供 $I$ 的电流源处理；没有给 $V_G$ / $V_T$ / source node，也无法唯一求 $R_{SS}$。

**🐈 防错提醒**：没图先看题目要你求什么；没写 tail resistance / source resistance 就不要自己加 $R_{SS}$。
