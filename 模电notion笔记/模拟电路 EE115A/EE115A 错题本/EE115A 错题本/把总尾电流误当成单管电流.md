# 把总尾电流误当成单管电流

错误总结: 差分对平衡时总尾电流会被两边平分；计算 gm、W/L 前先把单管 ID 写出来。
错误类型: 🔀 概念混淆, 🧮 计算失误

**❌ 错在哪**：把 tail current $I$ 当成单只 MOS 的 $I_D$，算 $g_m$ / $W/L$ 时忘记平衡状态下每边只有 $I/2$。

**✅ 正确做法**：题目给的 $g_m$ 是单管跨导，先用 $I_D=g_mV_{OV}/2=0.125\,\text{mA}$，再得 tail current $I=2I_D=0.25\,\text{mA}$，最后 $W/L=10$。

**🐈 防错提醒**：differential pair 题第一步先标清：这是 total tail current 还是 single-branch drain current。
